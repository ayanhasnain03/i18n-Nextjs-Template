# Next.js 15 Internationalization (i18n) with `next-intl`

This project demonstrates how to implement **internationalization (i18n)** in a Next.js 15 app using the [`next-intl`](https://github.com/amannn/next-intl) package. It includes locale-based routing, message translation, and a language switcher.

## Features

- Locale-specific routing (`/[locale]`)
- Translation message files (`en.json`, `de.json`, etc.)
- `next-intl` for loading and providing translations
- Language switcher component
- Example Hero component using translations with `useTranslations`
- Handling missing locales and fallback to 404

---

### Installation

Clone the repo:

```bash
git clone https://github.com/ayanhasnain03/i18n-Nextjs-Template.git
```
Change directory into the project folder:

```bash

cd i18n-Nextjs-Template
```
Install dependencies:

```bash

npm install
# or
yarn install
```
Start the development server:
```
bash
npm run dev
# or
yarn dev
```
Open your browser at http://localhost:3000/en to see the app in English locale.
